# Lab 3.1 — Pattern Structure Documentation

## Summary
This task documents a reusable JavaScript utility function using the formal pattern structure described in Chapter 3 of *Learning JavaScript Design Patterns, 2nd Edition*.

## Selected Function

## Included Files

## Methodology
The work was completed by:
1. selecting a practical utility function
2. analyzing its problem, context, and solution
3. documenting it as a formal pattern
4. creating a working JavaScript example
5. evaluating the documentation using the “Well-Written Patterns” criteria from Chapter 3
