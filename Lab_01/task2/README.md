# Lab 01 - Task 2 - Provider Pattern

## Run the project

1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

## Purpose

This project demonstrates the Provider Pattern in React using Context API.
It solves the prop drilling problem by sharing user data, permissions,
and theme preferences across nested components.
